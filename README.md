Scope of the Project – Arogya Nidhi (Healthcare App) 
The scope of Arogya Nidhi is to develop an Android application that 
provides basic healthcare assistance using Generative AI. It focuses on 
improving access to health information and promoting preventive care. 
Provide instant health guidance through an AI chatbot 
Enable symptom analysis and suggest possible conditions (non
diagnostic) 
Offer personalized health tips (diet, fitness, lifestyle) 
Allow users to store and track health data (e.g., weight, BP) 
Include emergency support features (contacts, nearby hospitals) 
Ensure user-friendly interface for all age groups 
Support scalability, so features like doctor consultation or wearable 
integration can be added later 
Limitations (Important Scope Boundaries): 
Not a replacement for professional medical advice 
AI provides general guidance only, not prescriptions 
Depends on internet connectivity for AI features 
Requirements of the Project 
1. Functional Requirements 
User registration/login (optional but recommended) 
AI chatbot for health-related queries 
Symptom input and response generation 
Health tips and recommendations module 
Health record storage and tracking 
Emergency contact access 
Notifications/reminders (water, medicine, etc.) 
2. Non-Functional Requirements 
Usability: Simple and intuitive UI 
Performance: Fast response time for AI queries 
Security: Protect user health data (authentication, encryption) 
Reliability: Accurate and consistent app behavior 
Scalability: Ability to add future features 
3. Technical Requirements 
Android Studio (development environment) 
Programming language: Kotlin or Java 
GenAI API integration (for chatbot & analysis) 
Firebase / SQLite (database) 
Internet connectivity 
Basic UI/UX design tools 
4. Hardware & Software Requirements 
Android device (version 7.0 or above preferred) 
Minimum RAM: 2GB (for smooth usage) 
Stable internet connection
